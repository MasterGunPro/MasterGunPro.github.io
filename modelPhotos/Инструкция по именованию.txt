Конфигуратор подтягивает изображения по выбранному оружию, модели, цвету наиболее близко к выбранному клиенту.

Если есть фото с винтовкой клиента - отобразиться она,
если совпадает и цвет - то при выборе акссесуаров он увидит свою винтовку в цвете,
иначе отобразиться выбранное ложе в цвете 
при отсутствии фото ложе в цвете - черно белое фото,
если нет ложе в цвете - NoPhoto.jpeg.



Если подходящая картинка не обнаружена - отображает NoPhoto.jpeg

Для оптимальной работы сайта важно, чтобы фото имели вес как можно меньший. Это сильно влияет на скорость
загрузки контента на утройстве пользователя. Для этого необходимо следующее:

1 - Фотографии должны быть в формате jpeg
	размером: максимальный по наименьшей стороне - 1000 пикселей 
	объемом: не более 0,5МБ
2 - Сохраннение фото нужно в формате для WEB в Фотошопе комбинацией клавиш Ctrl-Shift-Alt-S
3 - Сжать фото после сохранения для WEB через сервис https://tinypng.com/

Для каждой модели ложе (6 моделей) необходимо создать:
черно-белое фото <sku модели>.jpeg, например MG-01.jpeg

В цветах (16 цветов) <sku модели>_<sku цвета>.jpeg, например MG-01_C_150.jpeg




По мере появления вариантов моделей с винтовками желательно:

добавить черно-белое фото оружия с ложе <sku модели>_<sku оружия>.jpeg, например MG-01_R_BR8.jpeg

добавить цветное фото оружия с ложе <sku модели>_<sku цвета>_<sku оружия>.jpeg, например MG-01_C_145_R_BR8.jpeg
