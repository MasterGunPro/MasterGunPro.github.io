Зайдите в аккаунт github
Перейдите на страницу additionals. Например для копии сайта это https://github.com/MasterGunPro/MasterGunPro.github.io/blob/master/btrMastergunCopy/additionals.js

Найдите строки

var configuration = [
					{
						products: [
							//Общие аксессуары
							
Для общих аксессуара

вы можете добавить его, указав цену

{ sku: "D_108", price: "1500" },

Для аксессуара, который имеет единую ценну, но доступен для моделей MG-02, MG-03 добавьте его в общие аксессуары

{ sku: "D_x", price: "1000", visible:false },

А ниже укажите видисомсть данного аксессуара для моделей, добавив либо в уже существующий раздел

{
						model: "MG-02",
						products: [
						
Либо если такого раздела нет, добавьте

{
						model: "MG-03",
						products: [
						]
},

После этого в products добавьте строку меняющую видимость аксессуара
{ sku: "D_x", visible: true },

Если вам еще нужно переопределить цену (она отличается от общей) для данного аксессуара, укажите цену

{ sku: "D_x", visible: true, price: 1100 },

Перейдите в низ страницы

Введите комментарий что вы меняли

Нажмите Commit changes.

Через 1 минуту изменения будут доступны на сайте. Рекомендую обновить через Ctrl+F5
